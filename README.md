# 🌊 深海钢琴 - Piano in the Deep Cosmos

一个沉浸式的3D深海钢琴演奏体验，结合了梦幻的视觉效果和智能音乐功能。

## ✨ 项目特色

### 🎹 贝壳钢琴系统
- **49个音符**：覆盖C2-C6完整音域
- **贝壳造型**：每个琴键都是精美的贝壳设计
- **键盘映射**：支持完整键盘演奏
- **音效系统**：基于Tone.js的高品质音频

### 🐋 3D深海场景
- **游动鲸鱼**：85000+粒子构成的巨型鲸鱼，支持胸鳍拍打和脊椎波动
- **49只水母**：每个音符对应一只水母，演奏时会发光游动
- **Bloom后处理**：专业级辉光效果，营造梦幻氛围
- **动态相机**：自动环绕视角，沉浸式体验

### 🤖 AI智能功能
- **AI音乐生成**：支持多种风格和调性
- **智能助手小娜**：音乐理论指导和使用帮助
- **自动演奏**：AI生成旋律的自动播放
- **录制回放**：支持演奏过程录制和历史回放

### 🐟 鱼形界面设计
- **钢琴鱼**：键盘映射指南，钢琴键纹理
- **智能鱼**：AI音乐助手，电路板纹理
- **摄影鱼**：录制助手，胶片纹理
- **音乐鱼**：智能助手，五线谱纹理

## 🎮 使用指南

### 键盘映射
| 音区 | 键盘区域 | 音符范围 | 视觉层次 |
|------|----------|----------|----------|
| 高音区 | 数字行 `1-0` | C5-C6 | 近景水母 |
| 中音区 | QWERTY行 | C4-B4 | 中近景水母 |
| 中低音区 | ASDF行 | C3-B3 | 中景水母 |
| 低音区 | ZXCV行 | C2-B2 | 远景水母 |

### 功能按钮
- **🐟 钢琴鱼**：查看详细键盘映射指南
- **🐟 智能鱼**：音乐自动演奏
- **🐟 摄影鱼**：录制演奏过程和历史回放
- **🐟 音乐鱼**：智能助手小娜，获取帮助

### AI音乐生成
1. 点击智能鱼按钮打开AI助手
2. 选择音乐风格：古典、环境音乐、爵士、民谣、现代
3. 选择调性：C大调、G大调、A小调等
4. 点击"生成旋律"或"生成和弦"
5. 使用"自动演奏"功能播放生成的音乐

### 录制功能
1. 点击摄影鱼按钮打开录制助手
2. 点击"开始录制"
3. 在浏览器弹窗中选择"此标签页"
4. 勾选"分享音频"选项
5. 开始演奏，系统会自动录制
6. 录制完成后可在历史记录中回放

## 🎨 技术实现

### 核心技术栈
- **Three.js 0.160.0**：3D渲染引擎
- **Tone.js 14.7.77**：Web音频合成
- **WebGL**：硬件加速渲染
- **Web Audio API**：实时音频处理

### 渲染系统
```javascript
// 主要系统模块
- WhaleSystem: 鲸鱼粒子系统（85000+粒子）
- JellyfishSystem: 水母系统（49只，每只800+粒子）
- ShellPianoSystem: 贝壳钢琴系统
- BubbleEffectsSystem: 泡泡效果系统
- ScreenRecordingSystem: 录制系统
- AIMusicSystem: AI音乐生成系统
```

### 后处理效果
- **Bloom辉光**：多通道模糊和合成
- **粒子系统**：自定义着色器和纹理
- **动态光照**：实时阴影和反射
- **景深效果**：四层水母布局

### 音频系统
```javascript
// 音频配置
- 采样率: 44.1kHz
- 音色: 钢琴合成器
- 效果器: 混响、延迟、滤波
- 动态范围: 支持力度感应
```

```javascript

        { radius: 75, yOffset: -8, scale: 0.8 },  // 中景
```

## 🐛 故障排除

### 常见问题

**Q: 页面加载缓慢或卡顿**
A: 检查网络连接，确保Three.js和Tone.js库正常加载。可以打开开发者工具查看控制台错误。

**Q: 没有声音**
A: 确保浏览器允许音频播放，点击页面任意位置激活音频上下文。

**Q: 3D效果不显示**
A: 检查浏览器是否支持WebGL，更新显卡驱动程序。

**Q: 录制功能无法使用**
A: 确保浏览器支持Screen Capture API，在录制时选择正确的标签页和音频选项。

**Q: AI功能无响应**
A: AI功能为本地算法，无需网络连接。如果无响应，请刷新页面重试。

### 性能优化建议
- 关闭其他占用GPU的应用程序
- 使用Chrome浏览器获得最佳性能
- 降低浏览器缩放比例到100%
- 确保电脑有足够的内存（推荐8GB+）
